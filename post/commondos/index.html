打开CMD方式

  3. 文件夹地址栏输入cmd

常用Dos命令

基本操作

#清理屏幕 cls	(clear screen)
#打印当前位置 pwd		(print work directory)
#退出 exit

文件夹操作

#盘符切换 E:
#查看当前目录下所有文件  dir
#切换目录 cd	(change directory)
	cd /d 可以随意切换到目录下
	（\是和盘符相关）
	（/是和路径符相关）
#返回上一层 cd..
#文件夹 
 创建	md 目录名	（make dir）
 删除	rd 目录名
#文件 
 创建	cd> 文件名
 删除	del 文件名
#当前文件夹内容列表 
	ls -l	(list file)
